⚙️ Getting Started
Prerequisites
Make sure you have Node.js installed on your machine.

Installation & Setup
Clone the repository:

Bash
git clone https://github.com/thinh204/e-commerce.git
cd e-commerce
Install dependencies:

Bash
# For backend
cd backend
npm install

# For frontend
cd ../frontend
npm install
Running the application:

Bash
# Start the backend server
cd ../backend
npm run dev
